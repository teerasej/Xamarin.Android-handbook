
# การสร้างโปรเจค Xamarin.Android

เลือก **Create a new project**
(หรือจากเมนู **File > New Project**)

<img width="977" alt="2021-04-17_13-13-16" src="https://user-images.githubusercontent.com/85179/115103962-b4ec9a80-9f7f-11eb-92af-33d15ef10c59.png">


เลือก filter ดังนี้
1. **Language**: C#
2. **OS**: Android
3. **Platform**: Mobile
4. เลือก **Android App (Xamarin)**
5. กดปุ่ม **Next**

<img width="971" alt="2021-04-17_13-13-35" src="https://user-images.githubusercontent.com/85179/115103968-b74ef480-9f7f-11eb-9298-d00fb0f30d5a.png">

ระบุชื่อโปรเจค และที่อยู่สำหรับเก็บโปรเจค ให้เรียบร้อย

<img width="968" alt="2021-04-17_13-13-50" src="https://user-images.githubusercontent.com/85179/115103977-ba49e500-9f7f-11eb-8128-fe6a39c8d05a.png">

1. เลือก Template เป็น **Blank App**
2. และเลือก **Minimum Android Version** เป็น Android 6.0
3. กดปุ่ม OK
