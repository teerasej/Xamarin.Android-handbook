
# สร้าง Class สำหรับข้อมูล User 

## 1. การใช้งานเว็บแอพในการ gen class

- นำข้อมูล json มาจาก [https://randomuser.me/api/?results=50](https://randomuser.me/api/?results=50) มาใช้ 
- สร้าง code จาก Json ด้วย [QuickType.io](https://app.quicktype.io/) 
  - บนหน้าเว็บกำหนดชื่อ Class เป็น `UserRemoteModel`
  - กำหนด namespace เป็น `<ชื่อ namespace Project>.Model`

## 2. การสร้าง Class ในโปรเจค 
  - สร้างโฟลเดอร์ชื่อ `Model` ไว้ในโปรเจค
  - กำหนด class ชื่อ `UserRemoteModel` 
  - วางโค้ดที่ได้จาก QuickType.io ลงไป
  - แก้ไข `Title` Property เพราะบางครั้งข้อมูลเป็นค่าว่าง
